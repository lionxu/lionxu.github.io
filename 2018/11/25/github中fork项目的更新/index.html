github中fork项目的更新

我们在浏览GitHub的时候，如果遇到自己喜欢的项目，经常会把该项目的源码fork到自己的repository中去。

但是，这里存在一个问题，就是我们如何把被fork的项目后续代码的更新同步到我们自己的repository中去呢。

三、拉去被fork项目的最新代码
